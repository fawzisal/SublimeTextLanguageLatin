Sublime Text - Latin Language
============================

Latin Spelling Dictionary for Sublime Text 3

Installation
------------

### Sublime Package Control

  1. Install the following package : "Language - Latin"
  2. Choose the language (View > Dictionary > Language - Latin > la_LA)
  3. You are done !
